O que são Clients?

Um cliente é um computador ou programa configurado para utilizar os serviços oferecidos pelo servidor. Ele se conecta ao servidor pela rede. O cliente é responsável pela apresentação da interface do usuário. Por exemplo, quando você acessa uma página da web, seu computador (cliente) faz solicitações ao servidor para obter as páginas armazenadas nele.

O que são Servers?

Um servidor é um computador ou programa configurado em uma rede para oferecer serviços. Ele processa solicitações e fornece recursos ou informações para os clientes. Exemplos de servidores incluem servidores web (que hospedam sites), servidores de e-mail, servidores de arquivos, etc. O servidor lida com a lógica empresarial e o gerenciamento do estado. É o ponto central que atende às requisições dos clientes.

Como é uma estrutura básica de um HTML?

A estrutura do HTML inicia com a tag html e dentro dela ha as tags head e body. Na tag head é incluida a tag tittle onde é colocado o título da página. Já na tag body é colocado toda as outras tags para obter a pagina web.

Entendendo a diferença entre listas ordenadas e não ordenadas

Para as listas ordenadas é usada a tag ol, nela cada item é organizado em um indice em ordem crescente. Já nas listas não ordenadas é usada a tag ul, e cada item possui um pontinho no lugar de um número. Para adicionar itens nas duas listas é usada a tag li
